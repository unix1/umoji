# umoji

Search and compose emoji. The message lives in the URL path, so you can share links like `/🎉🔥`.

- Type to search · click to add · paste emoji into the message
- **Copy** / **Clear** the composed message
- Back/forward browser history syncs with the message

## Develop

```bash
npm install
npm run dev
```

## Deploy

Static assets on Cloudflare Workers:

```bash
npm run deploy
```
